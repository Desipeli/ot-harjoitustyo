## Asennusohjeet

1. Pura tiedosto haluamaasi hakemistoon
2. Suorita ``` poetry install ```

## Käyttö

1. Siirry komentorivillä pelin päähakemistoon
2. Suorita ``` poetry run invoke start ```
3. Pelaa!
